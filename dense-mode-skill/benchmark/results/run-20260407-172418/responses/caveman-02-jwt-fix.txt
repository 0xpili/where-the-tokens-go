The most common causes of "token expired" on a fresh token are:
- **Clock skew**: server time is ahead, so `exp` is already in the past from the verifier's perspective
- **Wrong unit**: expiry set in seconds but passed as milliseconds (or vice versa)
- **`iat` vs `exp` bug**: token created with `exp = iat - N` instead of `iat + N`
- **Timezone offset**: naive datetime used without UTC
